# Types Overview

## Built-in types

Joelang has a rich set of types:

- [Arrays](array.md)
- [Bools](bool.md)
- [Numbers](number.md)
- [Objects](object.md)
- [Paths](path.md)
- [Regexes](regex.md)
- [Strings](string.md)
- [Tuples](tuple.md)

## Custom types

You can create a custom type by defining a class.

```
class Foo {}
abstract class Bar {}
```
